# RF 網頁 spec

此文件為建構 RF 攻略資訊網站的各項功能

## 架設工具
1. python 
2. 使用 github pages 進行 deploy
3. 可透過 commit 來更新部分攻略資料


## 網頁內容
### 城鎮資訊
1. 名稱
2. 主權地
3. 劇情標題
4. 上/下一個城鎮
5. 劇情通關獎勵(含首次通關)、掉落物機率
6. 本地援助 & 據點戰獲勝獎勵

### 新手教學
1. 透過 md 架構來撰寫並分類，可支援 commit 更新

### 角色資訊
1. 角色名稱
2. 職業
3. 陣營
4. 生平
5. 天賦表 & 最大天賦
6. 角色圖片
7. 符文類型
8. 最大等級
9. 每等提升ATK&HP數值 (可滑動來看每一等的數值)

### 戰力計算器
1. 參考戰力計算網頁，可以自由編隊進行戰力計算

### 查詢相關
1. 角色查詢: 可以透過輸入帳密來查詢已擁有角色的詳細資料、目前等級/可提升最高等級，裝備符文等詳細資訊
2. 庫存查詢: 查詢目前所有道具的庫存、以及各符文等級&裝備者
3. 抽獎歷史查詢: 查詢過往抽卡紀錄，統計機率以及分佈

### 國策資訊
可以查看各陣營當前國策，以及套用到特定隊伍上的加成
